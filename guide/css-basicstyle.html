basicstyle.css

概要

とにかく作業効率を上げるために作られた、JRT案件特化型リセット系CSSと制作に関するスタイルのセットです。

構成

scssで作られています。

/assets/css/basicstyle.css

書き出されたCSS

/assets/scss/basicstyle.scss

インクルードでまとめられたSCSS、これで変数なんかをいじれる。

/assets/scss/basicstyle

basicstyle.scssにインクルードされるファイルがまとめられたディレクトリ。

/assets/scss/basicstyle/_reset.scss

"HTML5 Reset Stylesheet"をベースに<span>や<a>なんかの、"font-weight: nomal;"を"font-weight: inherit;"とかにしています。

あと、最初の方に"box-sizing: border-box"をすべてのタグに適用しています。

*,
*::after,
*::before {
	box-sizing: border-box;
	-webkit-box-sizing: border-box;
	-moz-box-sizing: border-box;
}

.list-style--*

".list-style" から始まるスタイルはデフォルトのリストスタイルを設定したものです。
ただブラウザ依存が強いので全てがちゃんと表示されるかは確認が必要です。

ulやolにクラスをつければ、クラスをつけた子要素のliにスタイルが当てられます。孫要素には設定されませんので、都度設定してください。

リストの左マージンは2.0emで"list-style-position: outside;"になっています。マージンの調整は上書きするか、basicstyle.scssに変数を入れて書き出し直してください。

リストの左マージンと、リストマークの幅は2.0emです。値は上書きするか、basicstyle.scssに変数を入れて書き出し直してください。
